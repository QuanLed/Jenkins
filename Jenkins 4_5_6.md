**Lesson 4: Cách dùng CLI - Command Line Interface**

  Dùng CLI thì:
      - Easier
      - Faster
      - Consume less memory
      - Continous Integration
      
  Truy cập link Jenkins, kèm path sau: `/cli`
  
  Sẽ hiển thị danh sách các câu lệnh của Jenkins
  
  Download jenkins-cli.jar và đặt vào đúng folder
